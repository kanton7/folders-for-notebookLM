# 개인정보처리방침 —  NotebookLM용 폴더

최종 수정일: 2026년 6월 15일

NotebookLM용 폴더(이하 "확장 프로그램")는 사용자의 개인정보를 수집하거나 외부로 전송하지 않습니다. 이 방침은 확장 프로그램이 어떤 데이터를 다루고 어디에 저장하는지 설명합니다.

## 수집하지 않는 정보

확장 프로그램은 다음을 수집·저장·전송하지 않습니다.

* 노트북의 제목, 본문, 소스 등 내용
* 이름, 이메일, 계정 정보 같은 개인 식별 정보
* 사용 기록, 분석(애널리틱스), 광고 식별자

확장 프로그램에는 외부 서버가 없으며, 어떤 데이터도 제3자에게 제공하지 않습니다.

## 저장하는 정보와 위치

확장 프로그램은 사용자가 직접 만든 "정리 정보"만 다룹니다.

* 폴더 이름과 색상
* 노트북에 부여한 태그
* 즐겨찾기 표시
* 노트북과 폴더의 연결 관계(어떤 노트북이 어떤 폴더에 속하는지)

이 정보는 브라우저의 저장소(chrome.storage)에만 보관됩니다. 폴더 목록은 동기화 저장소에, 노트북별 정리 정보는 로컬 저장소에 저장됩니다. 동기화 저장소에 보관된 항목은 사용자가 같은 계정으로 로그인한 Chrome 사이에서 Google이 제공하는 방식으로만 동기화되며, 확장 프로그램 개발자는 이 데이터에 접근할 수 없습니다.

## 권한을 사용하는 이유

* **저장소(storage)**: 위의 정리 정보를 브라우저에 보관하기 위해 사용합니다.
* **호스트 권한(https://notebooklm.google.com)**: NotebookLM 페이지에서만 노트북 목록을 읽고 정리 화면을 표시하기 위해 사용합니다. 다른 웹사이트에는 접근하지 않습니다.

## 데이터 삭제

정리 정보는 사용자의 브라우저에 있습니다. 확장 프로그램을 삭제하거나 브라우저의 확장 프로그램 데이터 삭제 기능을 사용하면 함께 지워집니다. 폴더 삭제 기능으로 개별 폴더를 지울 수도 있으며, 이때 노트북 자체는 삭제되지 않습니다.

## 제3자 제공

없습니다. 확장 프로그램은 데이터를 외부로 보내지 않으므로 제3자와 공유할 데이터가 존재하지 않습니다.

## 상표 안내

이 확장 프로그램은 Google과 무관한 비공식 도구입니다. NotebookLM과 Google은 Google LLC의 상표입니다.
